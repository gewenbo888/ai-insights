# AI Insights

A live feed of AI news and analysis, aggregated from trusted sources.

[![Live Demo](https://img.shields.io/badge/demo-ai--insights--beta.vercel.app-0ea5e9)](https://ai-insights-beta.vercel.app)

## Overview

A Next.js site that pulls AI-related articles from curated RSS feeds, categorises them, and presents them in a scrollable insights grid with a running ticker of the latest headlines. Content revalidates every 30 minutes to stay fresh.

## Design & Tech

- **Stack:** Next.js 14, React 18, Tailwind CSS
- **Data:** `rss-parser` pulling from multiple AI-focused feeds
- **Deployment:** Vercel
